# Cabin Rental Website

A simple, visually appealing static website for cabin rentals featuring property photos, descriptions, and contact information—without online booking functionality.

## Features

- Responsive design for all devices
- Hero section with call-to-action
- About section describing the rental experience
- Responsive image gallery with hover effects
- Contact form with client-side validation
- Footer with copyright and social links

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (for form validation and dynamic year)

## Setup

1. Clone the repository
2. Open `project/index.html` in a web browser
3. For development, consider using a local server (e.g., `python -m http.server 8000`)

## Customization

- Replace images in `/project/images/` with your own
- Update colors in `project/css/styles.css` by modifying CSS variables
- Update text content in `project/index.html`
- Add more gallery items by duplicating `.gallery-item` elements

## Deployment

This site can be deployed to any static hosting service:
- Netlify (drag and drop folder)
- Vercel
- GitHub Pages
- AWS S3 + CloudFront
